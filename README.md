# Sugarbox
Sugarbox is an Amstrad CPC/CPC+ Emulator

Currently emulated : 
- 464
- 664
- 6128
- 464 +
- 6128 +
- GX 4000

Support the following disk format :
- dsk & edsk
- HFE
- IPF
- CT-Raw
- SCP
- RAW Kryoflux

Support the following tape format : 
- CDT
- TAP
- CSW ( both 1.1 and 2.0)
- VOC
- WAV

Also support CDT (cartridge) format.
